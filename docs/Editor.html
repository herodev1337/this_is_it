# <Editor setter1 setter2 getter />

Main Editor Component.

There is a main div holding both editor instances the rest is managed by AceEditor. We only supply 2 things.

2 Setters strings which will go into the 'value' property of both editor instances. These will be watched and updated by react when we call the setText1/2 functions somewhere else in the code. These allow us to control what's going on in the editors.

1 Getter function which will go into the 'onChange' event on the first/main editor. This function gets called when the user types something into the main editor. The value it supplies will be all the text in the main editor.

PropTypes:
Name Type Required Description Default
setter1 string Yes Main editors text that should be shown.
setter2 string Yes Second editors (output) text that should be shown.
getter func Yes Main editors onChange function, should accept a string parameter. Get's called when something gets typed in the main editor.
